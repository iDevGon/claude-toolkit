# claude-toolkit

개인용 Claude Code 플러그인 마켓플레이스

## 설치

```bash
/plugin marketplace add iDevGon/claude-toolkit
```

## 플러그인 목록

| 플러그인 | 설명 | 스킬 |
|---------|------|------|
| `smart-commit` | 스마트 커밋 & 푸쉬 | `commit`, `commit-push` |
| `session-pipeline` | Jira 세션 컨텍스트 관리 | `session-start`, `session-clean` |
| `devops-tools` | 배포/인프라 도구 | `deploy-train`, `mcp-install` |

## 플러그인 설치

```bash
/plugin install smart-commit@claude-toolkit
/plugin install session-pipeline@claude-toolkit
/plugin install devops-tools@claude-toolkit
```
