# Session Clean — 티켓 메모리 정리

완료된 티켓의 세션 메모리 디렉토리를 삭제한다.

## 사용법

```
/session-clean PROJ-123
/session-clean PROJ-123 PROJ-456
```

## 실행 절차

### 1단계: 대상 확인

파라미터로 받은 티켓 번호에 해당하는 디렉토리가 존재하는지 확인한다.

```bash
ls -la ~/.claude/memory/<티켓번호>/ 2>/dev/null
```

디렉토리가 없으면 해당 티켓은 건너뛴다:
```
<티켓번호>: 메모리 없음 (스킵)
```

### 2단계: 삭제 전 확인

삭제 대상을 사용자에게 보여주고 확인을 받는다.

```
삭제 대상:
- <티켓번호>/context.md
- <티켓번호>/<레포명>/2026-04-02-14_30_15.md
- <티켓번호>/<레포명>/2026-04-02-15_00_00.md
- ...

총 N개 파일을 삭제합니다. 진행할까요?
```

### 3단계: 삭제

사용자가 확인하면 디렉토리 전체를 삭제한다.

```bash
rm -rf ~/.claude/memory/<티켓번호>
```

### 4단계: 결과 보고

```
<티켓번호> 메모리가 정리되었습니다. (N개 파일 삭제)
```

## 제약사항

1. **파라미터 없이 실행하면 안내 메시지만 출력한다.** 전체 삭제는 하지 않는다.
2. **삭제 전 반드시 사용자 확인을 받는다.**
3. **`~/.claude/memory/` 디렉토리 자체는 삭제하지 않는다.**
